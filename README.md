# Тестовое приложение для  Effective Moblie

Приложение отображает список курсов, полученных из API, позволяет сортировать их по дате публикации и добавлять курсы в избранное.

---

# Функциональность

### Авторизация
Экран входа с валидацией:

- Email вводится по маске `text@text.text`
- Кириллица в email запрещена
- Пароль принимает любые символы
- Кнопка "Вход" активна только при корректных данных

После успешной проверки осуществляется переход на главный экран.

### Кнопки перехода в социальную сеть

По нажатию:

- **VK** -> https://vk.com/
- **OK** -> https://ok.ru/

Открывается браузер.

Посмотреть скриншот можно в [pull request](https://github.com/IvanCRA/EffectiveMobile_test/pull/3)

---

### Главный экран

Отображается список курсов.

Функциональность:

- отображение курсов из API
- ограничение описания двумя строками
- сортировка по дате публикации
- добавление курса в избранное
- визуальное отображение рейтинга и даты публикации
- стеклянный бейдж рейтинга

Посмотреть скриншот можно в [pull request](https://github.com/IvanCRA/EffectiveMobile_test/pull/6)

---

### Избранное

Пользователь может добавить курс в избранное.

Особенности:

- избранные курсы сохраняются локально
- используется база данных **Room**
- список избранных отображается на отдельном экране
- состояние синхронизируется с главным экраном

Скриншот экрана и содержимое БД в [pull request](https://github.com/IvanCRA/EffectiveMobile_test/pull/7)

# Особенности реализации

- сортировка курсов по `publishDate`
- хранение избранного через **Room**
- реактивное обновление UI через **Flow**
- объединение данных API и локальной базы
- состояние избранного синхронизировано между экранами

Также на начальном этапе добавлена Ci, которая проверяет стиль кода через Ktlint и запуск билда приложения
